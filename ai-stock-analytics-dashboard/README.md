# 📊 AI Stock Analytics Dashboard

An interactive stock analysis dashboard built using Python and Streamlit.
This project visualizes stock data, calculates technical indicators, and predicts future prices using machine learning.

---

## 🚀 Features

* 📈 Real-time stock data using yfinance
* 🕯️ Candlestick charts with moving averages
* 📉 RSI (Relative Strength Index) indicator
* 🤖 Price prediction using Random Forest
* 💡 Buy/Sell signal based on trend

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Plotly
* Pandas & NumPy
* Scikit-learn

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

---

## 📊 Project Overview

This dashboard allows users to:

* Enter a stock ticker (e.g. AAPL, TSLA)
* View historical price trends
* Analyze momentum using RSI
* Get AI-based prediction for next-day price
